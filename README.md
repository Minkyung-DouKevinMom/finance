# 우리집 금융 관리 앱

가족 전체 자산을 한 곳에서 관리하기 위한 Streamlit + SQLite 앱입니다.

## 구성

```
finance_app/
├── app.py            # Streamlit 앱 (메뉴 7개)
├── db.py             # 스키마 정의 + DB 초기화
├── repository.py     # CRUD 헬퍼 (SQL은 모두 여기 모음)
├── requirements.txt
└── finance.db        # 실행 시 자동 생성되는 SQLite 파일
```

## 설치 및 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

처음 실행하면 `finance.db`가 같은 폴더에 생성되고, 자산구분(금융자산/부동산/실물자산)과
과목(보험·연금/예금·대출/주식·펀드/부동산/실물자산), 소유자(본인/배우자/공동)가 자동 시드됩니다.

## 첫 사용 순서

1. **계좌(상품) 관리 → 추가** 에서 모든 상품을 등록합니다.
   연금 상품은 **개시일·예상수령액**까지 입력해 두면 시뮬레이션에 바로 반영됩니다.
2. **월별 스냅샷 → + 새 시점 만들기** 로 이번 달 시점을 만든 뒤, 표에서 금액만 입력합니다.
3. 다음 달부터는 시점만 새로 만들고 금액만 갱신하면 됩니다 (상품 메타정보 재입력 불필요).

## 주의

- 모든 금액은 **만원** 단위로 통일했습니다 (엑셀과 동일).
- 잔액 기록이 있는 계좌를 삭제하면 데이터 보존을 위해 **비활성화**로 처리됩니다.
- 스냅샷 날짜는 항상 해당 월의 1일로 정규화됩니다.
